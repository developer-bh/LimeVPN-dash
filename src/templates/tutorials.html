How to Setup PPTP VPN on Windows

  • 1- Run Rasphone
  • 2- Press Win key + R to open Run window. Enter rasphone command and press OK
  • 3- Create a new network device by clicking on New
  • 4- Choose Workplace network
  • 5- In the Internet address, enter any one of the server mentioned under http://network.limevpn.com/
  • 6- Type in your LimeVPN login email and Password in their respective fields.
  • 7- Leave Domain blank and click Connect
  • 8- Wait for the process to finish and at the end you should be connected to the VPN
